# Цветовая схема приложения

## Основные цвета

Цветовая схема основана на зелёных тонах основного сайта:

### Primary (основные)
- **main**: `#2E7D32` - тёмно-зелёный (основной цвет)
- **light**: `#4CAF50` - светло-зелёный 
- **dark**: `#1B5E20` - очень тёмно-зелёный
- **contrastText**: `#ffffff` - белый текст

### Secondary (вторичные)
- **main**: `#4CAF50` - светло-зелёный
- **light**: `#81C784` - светлый зелёный
- **dark**: `#2E7D32` - тёмно-зелёный
- **contrastText**: `#ffffff` - белый текст

### Фоны
- **default**: `#f4f6fa` - светло-серый фон
- **paper**: `#ffffff` - белый фон для карточек

### Текст
- **primary**: `#212121` - основной текст
- **secondary**: `#757575` - вторичный текст

## Цвета бэджей задолженности (дизайнерские)

Для бэджей с информацией о задолженности используются яркие, дизайнерские цвета в стиле Material Design:

- **Общая задолженность**: `#2196F3` - яркий синий Material Design
- **ПДЗ (просроченная)**: `#E53935` - насыщенный красный Material Design
- **Подошедшие платежи (сегодня)**: `#FB8C00` - яркий оранжевый Material Design
- **Не просроченная задолженность**: `#43A047` - насыщенный зелёный Material Design

## Применение

Тема применяется через `ThemeProvider` в `main.tsx` и включает кастомные стили для:
- AppBar (зелёный фон)
- Button (зелёные кнопки)
- IconButton (hover эффекты)
- Chip (зелёные теги)

## Использование в компонентах

```tsx
import { useTheme } from '@mui/material/styles';
import { debtColors } from '../theme/theme';

const theme = useTheme();
const primaryColor = theme.palette.primary.main;
const debtColor = debtColors.all; // для бэджа общей задолженности
```
